# Installation

Install Bantai using your preferred package manager:

```package-install
@bantai-dev/core zod
```

<Callout>
[zod](https://zod.dev/) is a peer dependency and must be installed separately.
</Callout>

## Next Steps

Once installed, check out the [Quick Start](/docs/getting-started/quick-start) guide to create your first policy.
